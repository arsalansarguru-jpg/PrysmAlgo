# PrysmAlgo — Institutional Fintech Website

Premium institutional-grade algorithmic trading website built with Next.js 15, TypeScript, Tailwind CSS, Framer Motion, and Shadcn UI.

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
npm start
```

## Pages

| Route | Description |
|-------|-------------|
| `/` | Home — 12 conversion-optimized sections |
| `/performance` | Performance dashboard with charts |
| `/risk-management` | Institutional risk framework |
| `/about` | Company story, team, timeline |
| `/blog` | Research articles with categories |
| `/contact` | Contact form, WhatsApp, Calendly |
| `/apply` | Multi-step investor application |
| `/privacy-policy` | Privacy policy |
| `/terms` | Terms of service |

## Configuration

Update contact details, WhatsApp link, and Calendly URL in `src/lib/constants.ts`.

## Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide Icons
- Shadcn UI (Radix primitives)
- Recharts
